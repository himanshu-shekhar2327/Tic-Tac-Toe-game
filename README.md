# 🎮 Tic Tac Toe Game

A simple and fun **Tic Tac Toe** game built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- ✅ Clean and responsive UI
- 🧠 Logic to detect win and draw conditions
- 🔄 Restart functionality
- 👨‍👩‍👧‍👦 Two-player mode on the same device

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
